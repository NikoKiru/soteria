Frequently Asked Questions

Everything you need to know about Soteria.

Your data is protected by AES-256-GCM encryption, the same standard used by governments and banks. Your master password is processed through Argon2id, the winner of the Password Hashing Competition, using 128MB of memory to resist brute-force attacks. The encrypted vault file is unreadable without your master password‚Äîeven to Soteria.

There is no recovery option‚Äîthis is intentional. Without your master password, your vault cannot be decrypted by anyone, including us. This is the cornerstone of Soteria's security model: true encryption means true privacy. We recommend storing your master password in a secure physical location as a backup.

Soteria doesn't include cloud sync by design. However, you can manually transfer your vault file (~/.soteria/vault.json) to other devices using USB, encrypted email, or your own secure file sync solution. The vault is fully encrypted, so it's safe to move around.

Yes, completely free. Soteria is a personal project released as open source. There are no premium tiers, no subscriptions, no hidden costs. The source code is available on GitHub for anyone to inspect, modify, or contribute to.

Soteria officially supports Windows with a native executable. Since it's built with Python, it can also run on macOS and Linux by running from source. The browser extension works with Chrome, Edge, and Brave.
